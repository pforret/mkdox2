# D-0001 — Establish a decisions log

- **Status:** Accepted
- **Decided:** {CREATION_DATE}
- **Meeting:** [Kickoff](../meetings/2026/2026-04-22-kickoff.md)
- **Vote:** unanimous

## Context

We need a single place where standing decisions live, separate from the chronological [meeting minutes](../meetings/index.md). Without it, the same questions get re-debated every term.

## Decision

We will maintain a [Decisions log](index.md) on this site. Every formal decision is recorded with a short context, the decision itself, and a link back to the meeting where it was taken.

## Consequences

- New members can read the log and inherit the standing commitments.
- The [Secretary](../members/roles.md) is responsible for adding entries within _N_ days of each meeting.
- Superseding a decision requires a new entry that references the old one — the old entry is never edited, just marked **Superseded**.
