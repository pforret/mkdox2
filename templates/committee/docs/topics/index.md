# Topics

Standing themes the committee tracks. Different from [projects](../projects/index.md): a topic is ongoing; a project has an end date.

## What a topic page contains

- A short description of the theme.
- Standing commitments and policies (cross-linked to [Decisions](../decisions/index.md)).
- The list of [projects](../projects/index.md) under this topic.
- Open questions and recurring concerns.
- The topic lead is responsible for keeping the page current.

## Adding a new topic

1. Propose at a [meeting](../meetings/index.md).
2. If approved, log in [Decisions](../decisions/index.md).
3. Create a new page here, link from this index and from [Members](../members/index.md) for the lead.
