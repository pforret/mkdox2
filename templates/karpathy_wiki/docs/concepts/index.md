# Concepts

Ideas, theories, frameworks, methods — the abstract counterparts to [entities](../entities/index.md).

## Format

Each concept lives at `concepts/<slug>.md` with frontmatter:

```yaml
---
type: concept
sources: [sources/source-1.md, sources/source-2.md]
updated: YYYY-MM-DD
---
```

A typical concept page has sections for: definition, key claims (each with a source citation), related concepts, related entities, open questions, and a `## Connections` section.
