# Production workflow

How a signed engagement gets delivered.

## 1. Kickoff

- Internal kickoff with [team](../people/team.md): scope, owners, milestones
- External kickoff with the client: agree comms cadence, success criteria
- Log all decisions to [Decisions](../actions/decisions.md)

## 2. Delivery

- Daily progress in [Actions](../actions/index.md)
- Weekly status email using the [follow-up template](../templates/follow-up.md)
- Every client call → [transcript](../transcripts/index.md) → AI files actions

## 3. Handoff

- Final deliverables checklist
- Retrospective entry in [Decisions](../actions/decisions.md)
- Update [client](../people/clients.md) status to `delivered` or `retainer`
