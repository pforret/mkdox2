# Transcripts

Raw call transcripts land here. The AI processes them and files the output across the vault.

## Workflow

1. Call ends → transcription tool exports a `.md` or `.txt` file into this folder.
2. Ask the AI: _"Process the latest transcripts."_
3. The AI extracts:
    - **Summary** → appended to the matching [client page](../people/clients.md)
    - **Decisions** → appended to [Decisions log](../actions/decisions.md)
    - **Actions** → appended to [Action Tracker](../actions/index.md)
4. The transcript stays here as the immutable source of truth.

## Conventions

- Filenames: `YYYY-MM-DD-<client>-<topic>.md`
- Never edit a transcript after the fact — append a note to the client page instead.
