# Decisions Log

Append-only. Every meaningful decision the team makes lands here, with the why and the source. The AI files entries here when it spots a decision in a [transcript](../transcripts/index.md).

## Format

```
## [YYYY-MM-DD] short title
- **Decision:** what we agreed
- **Why:** the reason / context
- **Affects:** link to [client](../people/clients.md), [framework](../library/index.md), or person
- **Source:** link to transcript or chat session
```
