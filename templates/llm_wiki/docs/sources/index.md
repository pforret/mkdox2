# Sources

Catalog of every ingested source. One page per source.

## Format

Each source summary lives at `sources/<slug>.md` with frontmatter:

```yaml
---
type: source-summary
source_path: ../../raw/<filename>
ingested: YYYY-MM-DD
---
```
