# Entities

People, places, things, products, organisations — anything with a proper noun.

## Format

Each entity lives at `entities/<slug>.md` with frontmatter:

```yaml
---
type: entity
sources: [sources/source-1.md, sources/source-2.md]
updated: YYYY-MM-DD
---
```

A typical entity page has sections for: overview, key facts, related entities (with links), related concepts (with links), open questions.
